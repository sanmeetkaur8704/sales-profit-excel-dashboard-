This project presents an interactive Sales and Profit Dashboard created using Microsoft Excel. The dashboard analyzes business data to provide insights into sales performance, profit trends, customer contribution, and regional distribution.

PROJECT FEATURES

* Total Sales and Total Profit KPIs  
* Sales by Product Category  
* Profit by Year comparison  
* Customer Count by Year  
* Top 5 Customers by Profit  
* Sales by State (Map View)  
* Monthly Sales Trend Analysis  
* Interactive filters for Category, Year, and Month
